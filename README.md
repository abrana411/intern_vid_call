# Features:
1) Google Authentication
2) State Persistence
3) Video Meeting
4) Storing Past Meets History
5) Create Meet
6) Join an Existing Meet
7) All Meeting Functionalities (Share screen , Hand raise etc.)

# Tech Stack:
Flutter, Dart, Firebase
